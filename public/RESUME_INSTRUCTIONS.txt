RESUME PDF SETUP INSTRUCTIONS
==============================

To enable the "Download Resume" button on your portfolio:

1. Export your resume as a PDF file
2. Name it exactly: resume.pdf
3. Place it in this public/ folder
4. Delete this RESUME_INSTRUCTIONS.txt file
5. Commit and push to GitHub

The resume will be automatically downloadable from:
https://your-portfolio-url.vercel.app/resume.pdf

Note: Until you add your resume.pdf, the Download Resume button will not work.
